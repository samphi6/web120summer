THIS, IS, FLEXBOX!

With any luck, I’ll be able to master this skill and find different ways to design with efficiency and poise. Today, I’ve learned of something that may help me do just that. The Flexbox Layout Module creates a simple way to manipulate flexible responsive format without the use of the popular properties of position and float. This layout method is supported in the browsers Chrome, I.E., Firefox, Safari and Opera as of August 2015. Flexbox puts items in a single row or column. This layout put together will include the use of flex containers in which flex items are held. We can manipulate the design by arranging the containers either horizontally or vertically. Organizing the design in such a way is referenced as the main axis.

The benefit of these items inside these containers is in the ability to resize an item larger to fill unused white space or shrink so to work around overflowing. The children of flex containers are placed in columns or rows in line with the main axis. Flexbox makes our work in creating websites much easier with little code. By using .flex-container { display: flex; } we are enabling the container to become larger or smaller vertically, matching the height of the biggest item. This also tells the containers and items to resize equally. This also signals for items to be sized based on their width and be arranged from the left side of the container. .flex-container will be treated like a flex container and will treat children of .flex-container as flex items. So damn wonderful, I could kiss it! With that said, I am now on the path of becoming a flexbox master.
